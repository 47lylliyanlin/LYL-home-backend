## 🛠 技术栈总览

### 前端
- **HTML/CSS/JavaScript** - 基础网页技术
- **浏览器API** - MediaRecorder（录音）、Audio（播放）、localStorage（存储）

### 后端
- **Python 3.11** - 编程语言
- **FastAPI** - Web框架
- **FunASR** - 语音识别（阿里达摩院）
- **MeloTTS** - 语音合成
- **ChromaDB** - 向量数据库
- **Anthropic Claude API** - AI对话模型

### 部署（待定）
- **方案A：** VPS + Nginx + Supervisor
- **方案B：** Railway（后端） + Supabase（前端+数据库）
